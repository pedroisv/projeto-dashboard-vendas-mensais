#  **Dashboard Vendas Mensais**

## **Visão Geral**


Este projeto apresenta um **dashboard em Power BI** para análise do desempenho mensal de vendas de um e-commerce fictício.  
A solução permite:

- Comparar o **mês atual** com o **mês anterior**
- Avaliar as métricas principais: **Receita**, **Lucro**, **Quantidade Vendida** e **Ticket Médio**
- Identificar os fatores que mais influenciaram a variação mensal
- Analisar resultados por **categoria**, **produto** e **região**

---

##  **Tecnologias Utilizadas**

- **Power BI** — construção do dashboard e criação das medidas DAX  
- **PostgreSQL** — extração e tratamento dos dados  
- **DAX** — cálculos de indicadores e medidas dinâmicas
